# Groups

## List groups

`GET /api/v1/groups`

## Retrieve a group

`GET /api/v1/groups/{id}`

## Create a group

`POST /api/v1/groups`

## Update a group

`PUT /api/v1/groups/{id}`

## Upload a group avatar

`POST /api/v1/groups/{id}/avatar`

## Delete a group

`DELETE /api/v1/groups/{id}`

## Delete a group avatar

`DELETE /api/v1/groups/{id}/avatar`
