# Groups

## List groups

`GET /api/group`

## Retrieve a group

`GET /api/group/{id}`

## Create a group

`POST /api/group`

## Update a group

`PUT /api/group/{id}`

## Upload a group avatar

`POST /api/group/{id}/avatar`

## Delete a group

`DELETE /api/group/{id}`

## Delete a group avatar

`DELETE /api/group/{id}/avatar`
